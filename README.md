# trabalho-pln-visao
Trabalho prático de PLN e Visão Computacional desenvolvido em Python no Google Colab. Contém a implementação de Reconhecimento de Entidades Nomeadas (NER) e Análise de Sentimentos usando spaCy/TextBlob, além de Segmentação de Imagem por Limiarização de Otsu com OpenCV. Códigos e evidências inclusos.
